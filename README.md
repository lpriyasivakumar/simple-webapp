# simple-webapp
Testing languages for backend apps

App uses the public api `https://dog.ceo/dog-api/` to test backend api setup in different languages  
It compares backend apis in:  
- Go
- Python using flask
- Kotlin using Ktor
- Kotlin using Spring boot
